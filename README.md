## Estrutura da Aula (2h, sem pausa)

| Bloco | Duração | Conteúdo |
|-------|---------|----------|
| Markdown avançado | 15 min | Tabelas, estrutura, template |
| SPIRIT + cenários + workflow IA | 25 min | Secções, atribuição de grupos, uso de IA |
| Trabalho prático | 35 min | Redação em Markdown com auxílio de IA |
| Pandoc | 15 min | Conversão Markdown → Word |
| Word | 20 min | Estilos, índice automático |
| Referências + TPC + encerramento | 10 min | Zotero, próxima aula |
